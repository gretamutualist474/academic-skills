# Skill Collection

This folder is a collection root for multiple skills.

## Layout

- `skill-name/SKILL.md` is the required entry point for each skill.
- Optional bundled resources live beside `SKILL.md`:
  - `evals/`
  - `scripts/`
  - `references/`
  - `assets/`

## Conventions

- Use lowercase kebab-case for skill folder names.
- Keep each skill self-contained inside its own folder.
- Put only collection-level notes in this root `README.md`.
